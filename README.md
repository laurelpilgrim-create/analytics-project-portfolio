Modeling Diabetes Progression: A Comparison of Linear Regression, K-means, and PCA 

The goal of this project is to analyze the Scikit-learn Diabetes Dataset to evaluate how well different machine learning techniques 
can model or describe the data. This study applies Linear Regression, which is a supervised learning model to predict diabetes progression
and compares it with a K-means clustering with and without Principal Component Analysis (PCA), which is an unsupervised learning method. 
The objective is to determine which approach best fits the dataset. 
Based on performance metrics such as Mean Squared Error (MSE), R-squared, inertia, and silhouette score.
